# DarkChess

一個以 Vue 3 製作的暗棋小遊戲，提供雙人對戰與人機對戰，並支援棋盤樣式與棋子樣式切換。專案目前預設使用貓咪棋子主題，開局後即可在 8 x 4 棋盤上進行翻棋、移動與吃子。

## 功能特色

- 支援 `雙人` 與 `人機` 兩種遊戲模式
- 人機模式提供 `簡單`、`普通`、`困難` 三種難度
- 可切換 `基本棋盤` 與 `木質棋盤`
- 可切換 `傳統` 與 `貓咪` 棋子主題
- 首翻後自動決定紅黑陣營
- 支援暗棋基本規則：
  - 未翻開的棋子可先翻面
  - 已翻開的棋子可移動到相鄰空格
  - 可攻擊相鄰敵方棋子
  - 炮可隔一子直線攻擊
  - 兵、卒可吃將、帥，將、帥不可吃兵、卒

## 遊戲規則簡述

- 棋盤為 `8 x 4`
- 每回合只能進行一個動作：
  - 翻開一顆未揭示棋子
  - 移動一顆已揭示且屬於自己陣營的棋子
  - 吃掉可合法攻擊的敵方棋子
- 任何一方的棋子被全部吃光時，遊戲結束
- 人機模式中，電腦會在自己的回合自動思考並行動

## 操作方式

1. 選擇遊戲模式
2. 人機模式下可先選擇電腦難度
3. 選擇棋盤樣式與棋子樣式
4. 按下 `開始遊戲`
5. 依照畫面提示進行翻棋與移動

## 專案結構

- `index.html`：主畫面與 UI 結構
- `app.js`：遊戲邏輯、AI、規則判定與狀態管理
- `style.css`：視覺樣式
- `assets/pieces/cat/`：貓咪主題棋子圖檔

## 執行方式

這是純前端靜態專案，沒有額外建置步驟。

### 方式一：直接開啟

直接在瀏覽器開啟 `index.html` 即可。

### 方式二：使用靜態伺服器

如果你的瀏覽器對本機檔案載入有額外限制，可以用任意靜態伺服器開啟專案目錄，例如：

```bash
npx serve
```

或是使用 VS Code Live Server。
