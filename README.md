# DarkChess

DarkChess 是一個以 Vue 3 製作的暗棋小遊戲，提供快速開局的雙人對戰與人機對戰，並支援棋盤樣式、棋子樣式與 AI 難度切換。專案目前預設使用貓咪棋子主題，玩家可以在 8 x 4 的棋盤上翻棋、移動、吃子，享受輕量但完整的暗棋對戰體驗。

## 功能特色

- 支援 `雙人` 與 `人機` 兩種遊戲模式
- 人機模式提供 `簡單`、`普通`、`困難` 三種難度
- 可切換 `基本棋盤` 與 `木質棋盤`
- 可切換 `傳統` 與 `貓咪` 棋子主題
- 首翻後自動決定紅黑陣營
- 介面提供目前回合、先手結果、陣營分配與剩餘棋子數量提示
- 支援暗棋基本規則：
  - 未翻開的棋子可先翻面
  - 已翻開的棋子可移動到相鄰空格
  - 可攻擊相鄰敵方棋子
  - 炮可隔一子直線攻擊
  - 兵、卒可吃將、帥，將、帥不可吃兵、卒

## 規則說明

- 棋盤為 `8 x 4`
- 每方共有 `16` 顆棋子，合計 `32` 顆
- 紅黑棋子會在首翻後才決定分配給玩家或電腦
- 雙人模式由玩家一先手；人機模式則會隨機決定玩家或電腦先手
- 每回合只能進行一個動作：
  - 翻開一顆未揭示棋子
  - 移動一顆已揭示且屬於自己陣營的棋子
  - 吃掉可合法攻擊的敵方棋子
- 一般棋子只能上下左右移動一格
- 一般棋子吃子時也只能攻擊上下左右相鄰的敵方棋子
- 炮的移動規則：
  - 沒有隔子時，可直線移到相鄰空格
  - 直線上若剛好隔一顆棋子，則可吃第二顆棋子
  - 炮不能以一般方式直接吃相鄰棋子
- 兵、卒可以吃將、帥
- 將、帥不能吃兵、卒
- 其餘棋子的吃子強弱依照棋子等級判定
- 任何一方的棋子被全部吃光時，遊戲結束
- 若某一方沒有任何合法動作，另一方直接獲勝
- 人機模式中，電腦會在自己的回合自動思考並行動

## 操作方式

1. 選擇遊戲模式
2. 人機模式下可先選擇電腦難度
3. 選擇棋盤樣式與棋子樣式
4. 按下 `開始遊戲`
5. 依照畫面提示進行翻棋與移動

## 專案結構

- `index.html`：主畫面與 UI 結構
- `app.js`：遊戲邏輯、AI、規則判定與狀態管理
- `style.css`：視覺樣式
- `assets/pieces/cat/`：貓咪主題棋子圖檔

## 執行方式

這是純前端靜態專案，沒有額外建置步驟。

### 方式一：直接開啟

直接在瀏覽器開啟 `index.html` 即可。

### 方式二：使用 repo page

如果你想把它掛到 repo 的子頁，直接開啟 `page/index.html`。

### 方式二：使用靜態伺服器

如果你的瀏覽器對本機檔案載入有額外限制，可以用任意靜態伺服器開啟專案目錄，例如：

```bash
npx serve
```

或是使用 VS Code Live Server。

## 備註

- 專案目前透過 CDN 載入 Vue 與 Bootstrap
- 預設棋子主題為 `貓咪`

- 人機模式的 AI 會根據難度做基本策略選擇，包含翻棋、吃子與風險評估
